# First Things Planner

A premium-feeling LDS-centered progressive web app for ordering daily goals and tasks by significance while keeping scripture study and prayer first.

## Features

- Faith anchors keep scripture study and prayer ahead of every other task.
- Significance-based sorting for user-entered goals and tasks.
- Guided planning prompts when the user needs goal ideas.
- Light and dark modes, motion, and a clean sudoku/crossword-inspired grid aesthetic.
- Local persistence with `localStorage` and offline support through a service worker.

## Development

```bash
npm install
npm run build
npm start
```
